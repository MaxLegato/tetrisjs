Game uses compressed version of solution script file called tetris_comp.js.

PRESS ENTER
Score: 0
Use arrows to move piece around Press A to rotate piece left (-90°) Press D to rotate piece right (+90°) Press SPACEBAR to drop piece Press ENTER to start game
